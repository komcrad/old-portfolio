In June 2014 Konrad was hired by Red Lion as an intern. Konrad was tasked to learn how to test the cellular products made by the branch in St. Louis, design a system that could test these products, and then write tests for the products.

After being a proof of concept java desktop application, Glados transformed into a web app capable of many useful functions.

Glados tests cellular and network devices through a web GUI and console access over a network. This gives the system great flexibility in testing the device both as a human tester would, and in much more tedious ways that a human simply could not test every release cycle.
